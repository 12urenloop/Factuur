# Factuur
Onze huisgemaakte factuurgeneratie-applicatie.

## Hoe start ik?

1. Installeer:
    - Ruby 2.5.0
    - Bundler
    - Node (met `npm`)
    - Sqlite
2. `bundle install`
3. `npm install`
4. `rails db:setup`
5. `rails s`
6. Browse to `http://localhost:3000`
